COMO ABRIR O JOGO

1. Extrai o ZIP completo.
2. Abre a pasta extraída.
3. Abre o ficheiro index.html.

Importante:
Não tires o index.html para fora da pasta.
Ele precisa das pastas js, css e assets ao lado dele.

O progresso fica guardado apenas no navegador/dispositivo onde jogares.
Se o navegador bloquear o som no arranque, clica uma vez no jogo ou no menu.
