# Sons V24

Esta versão já inclui ficheiros de som reais em WAV na pasta:

`assets/audio/`

## Músicas incluídas

- `menu-theme.wav` — menu inicial
- `map-peace.wav` — cidade em paz
- `map-dark.wav` — mapas sombrios
- `battle-theme.wav` — batalhas normais
- `boss-theme.wav` — bosses

## Efeitos incluídos

- `click.wav` — botões/menu
- `light-attack.wav` — ataque de luz
- `dark-hit.wav` — impacto sombrio
- `victory.wav` — vitória
- `heal.wav` — cura

## Como trocar por sons melhores

Se depois quiseres usar sons profissionais, basta substituir os ficheiros dentro de `assets/audio/` mantendo os mesmos nomes.

Também podes mudar os nomes em:

`js/config/audio-config.js`

## Nota

Os sons desta versão são gerados de forma simples, só para o jogo já ter ambiente sonoro.
A estrutura está pronta para trocar por músicas/efeitos melhores mais tarde.
