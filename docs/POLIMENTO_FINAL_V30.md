# V30 — Polimento final para testers

Esta versão mantém os atalhos de boss para desenvolvimento, mas melhora a experiência normal do jogo.

## Alterações

- O botão Menu agora pausa a atualização do jogo enquanto o menu está aberto.
- Ao voltar ao menu, a música volta para o tema do menu.
- Foi adicionado aviso visual de “Nova missão”.
- Foi adicionado aviso visual de “Progresso guardado”.
- As batalhas principais têm mensagens de abertura mais narrativas.
- O resultado das lutas principais deixa de ser sempre apenas “Vitória”:
  - Kraidus destruído
  - Nilzin recua
  - A sombra recua
  - Lurei enfraquecido
  - Rainha das Sombras vencida
- Foram reforçadas falas de NPCs por fase da história.
- Foram reforçadas falas do final com Lurei, Divan, Cavaleiros e aldeões.
- Foram melhoradas algumas descrições de missão.
- O menu inicial ganhou uma linha rápida com os controlos.

## Nota

Os atalhos de boss continuam disponíveis para desenvolvimento e testes. A ideia é remover/esconder totalmente apenas na versão final pública.
